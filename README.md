# Vue Linters

Шаблон для настройки линтеров в проекте на базе Vue 3 и Vite.

## Как использовать

1. Клонируйте репозиторий
2. Установите зависимости `npm install`
3. Запустите проект `npm run dev`

## Линтеры, установленные в проекте

- ESLint с настройками для Vue
- Prettier с настройками для форматирования кода
- TypeScript с настройками для типизации
- Husky с настройками для pre-commit и pre-push хуков
